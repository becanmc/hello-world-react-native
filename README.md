# Hello, React Native!
Esse é meu primeiro projeto em React Native.

O projeto consiste em aprender estruturas, componentes básicos e familiarizar com o framework.

Existe apenas uma (1) tela, onde o usuário inputa quantos Grammys a Adele já ganhou.

## Como rodar o projeto
1. Dê clone do repósitorio na sua máquina:
```bash
git clone https://github.com/becanmc/hello-world-react-native.git
```

2. Abra o projeto:
```bash
cd ./hello-world-react-native/project
```

3. Instale as dependências:
```bash
npm install
```

4. Rode o projeto:
```bash
npx expo start
```

5. Leia o QRCode com o celular (Necessário ter instalado o Expo Go) ou pressione as teclas que aparecem no menu do terminal.
